Installation

Choose your platform and follow the steps below to set up SSH GitHub Key Genie.

Windows (PowerShell)

  1. Clone the repository: 
    git clone https://github.com/adminscode/ssh-githubkey-genie.git
  2. Open PowerShell and navigate into the repo folder: 
    cd ssh-githubkey-genie
  3. Run the script: 
    .\new-github-ssh-key.ps1

Linux / macOS (Python)

  1. Clone the repository: 
    git clone https://github.com/adminscode/ssh-githubkey-genie.git
  2. Ensure you have Python 3 installed: 
    python3 --version
  3. Navigate into the repo folder: 
    cd ssh-githubkey-genie
  4. Run the script: 
    python3 new_github_ssh_key.py

Prerequisites

  • Git — required to clone the repository.
  • OpenSSH tools — for key generation and agent usage.
  • Clipboard utilities (Linux only, e.g. xclip or xsel) for automatic key copying.
  • GitHub Personal Access Token (optional) — required only if you want the script to upload your key automatically.

Verify installation

After installation, test by running:

ssh -T git@github.com

If everything worked, GitHub will greet you by your username.
